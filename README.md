# GeekStore — Frontend Test N1

O objetivo deste teste é avaliar sua capacidade de desenvolver layouts responsivos seguindo a abordagem mobile-first, além de propor e implementar interações não previstas explicitamente no design. Dito isso, no figma abaixo teremos somente o layout desktop e você deve construir o mobile com base no layout desktop.

## Escopo do projeto
Homepage de e-commerce para games, consoles e produtos geek.


## Responsividade

| Breakpoint | Comportamento |
|---|---|
| `< 768px` | Menu hamburguer + drawer mobile, topbar e nav ocultos e grid 2 colunas |
| `≥ 768px` | Header completo, busca visível, nav com dropdowns, grids 4 colunas |
| `≥ 1024px` | Hero maior, títulos maiores no banner mosaico |
